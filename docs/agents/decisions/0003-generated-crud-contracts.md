# 0003 - Generated CRUD Contracts

Baseline: current stable implementation.

## Decision

Standard REST and microservice CRUD contracts are generated by controller factories.

## Rationale From Code

`BaseController` creates REST list/search/detail/create/update/delete endpoints. `ClientController` creates message-pattern CRUD handlers. `ClientService` maps CRUD calls to microservice commands.

## Consequence

Feature modules can reuse consistent contracts and override or extend them with custom routes as needed.
